## 2. Identified Issues

### Critical Issues
1. **Hardcoded Windows Paths** (`gateway/gatekeeper_logic.py:10-11`)
   - `VAULT_PATH = r"d:\SME\data\grok_vault"`
   - `SIGNATURES_PATH = r"d:\SME\data\signatures.json"`
   - Fix: Use environment variables or relative paths

2. **Database Connection Management** (`gateway/nexus_db.py`)
   - Single SQLite connection, no pooling
   - WAL mode doesn't scale across Docker containers
   - Fix: Implement SQLAlchemy pooling or connection sharing

3. **Main Entry Mismatch** (`__main__.py`)
   - Imports reference `src.*` modules
   - But code runs from `gateway.mcp_server`
   - Fix: Align imports or consolidate entry points

### Medium Issues
1. **Inconsistent Rate Limiting** - Two independent limiters
2. **JWT No Refresh Token** - 24h hardcoded expiry
3. **Session History Limit** - Hardcoded 10 entries
4. **Import Blocker Gaps** - Missing `marshal`, `pickle` blocks
5. **Metrics Port Collision** - Port 8000 conflict

### Security Concerns
1. **Symlink Detection** - Only checks top-level, not nested
2. **SQL Injection Risk** - Some dynamic SQL construction
3. **Secret Rotation** - No mechanism for JWT secret rotation
