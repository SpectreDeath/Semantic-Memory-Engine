# SME/Lawnmower Man Forensic Analysis Workflow
## Target: Stephen Miller Stylometric Case

---

## Overview

This plan outlines a comprehensive workflow to run the SME tool suite against the Stephen Miller case files (`private/stephen_miller_case/`) to build a complete forensic profile.

---

## Workflow Phases

### Phase 1: Basic Stylometric Analysis
**Tool**: `src/sme/vendor/forensic_metrics.py`
- Calculate Entropy, TTR, Burstiness
- Compare against AI baseline
- Output signature vectors

### Phase 2: Signature Comparison
**Tool**: `src/analysis/comparisons.py`
- Compare against stored corpus signatures
- Compute Manhattan Distance
- Identify closest matches

### Phase 3: Rhetoric & Sentiment Analysis  
**Tools**: 
- `src/analysis/rhetoric.py` - Dehumanizing language detection
- `src/core/sentiment_analyzer.py` - Multi-perspective sentiment
- Detect "Character Tells" (vermin, parasites, etc.)

### Phase 4: NLP Processing
**Tool**: `src/core/nlp_pipeline.py`
- POS tagging
- Named Entity Recognition
- Lemmatization
- Dependency parsing

### Phase 5: Knowledge Graph
**Tool**: `src/analysis/knowledge_graph.py`
- Entity-relation extraction
- Semantic network visualization
- Central entity analysis

### Phase 6: Overlap Discovery
**Tool**: `src/analysis/overlap_discovery.py`
- Cross-document similarity
- Hidden connection detection

### Phase 7: Data Processing
**Tool**: `src/analysis/polars_forensics.py`
- Process corpus into Polars DataFrames
- Compute aggregations

---

## Implementation Steps

### Step 1: Import/Adapt Tools
Copy relevant tools to case working directory or run from SME:
```python
# Add SME src to path
import sys
sys.path.insert(0, "D:/GitHub/projects/SME/src")

from sme.vendor.forensic_metrics import calculate_cosine_delta, calculate_symmetrized_kl_divergence
from analysis.comparisons import SignatureCompare
from analysis.rhetoric import RhetoricAnalyzer
from core.sentiment_analyzer import SentimentAnalyzer
```

### Step 2: Run Basic Metrics
Calculate for each case file:
- TTR (Type-Token Ratio)
- Shannon Entropy
- Burstiness
- Average Sentence Length

### Step 3: Cross-Platform Comparison
Compare metrics across:
- Government documents (GOV)
- Social Media (SM)  
- Podcasts (POD)
- Historical (HIST)

### Step 4: Rhetoric Analysis
Apply custom lexicon to detect:
- Dehumanizing metaphors
- Bureaucratic distancing
- High-impact vocabulary clusters

### Step 5: NLP Analysis
Extract:
- Part-of-speech distributions
- Named entities (people, places, organizations)
- Key noun phrases

### Step 6: Knowledge Graph
Build graph of:
- Entities mentioned
- Relationships between them
- Topic clusters

### Step 7: Overlap Detection
Find semantic connections across:
- Different time periods (2002 vs 2025)
- Different platforms (formal vs social media)

### Step 8: Generate Final Report
Compile all results into enhanced forensic_report.md
